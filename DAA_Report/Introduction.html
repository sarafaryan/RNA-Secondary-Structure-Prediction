The problem statement involves finding all the pairs of complementary bases that match. Adenine(A) always matches with Uracil(U) while Cytosine(C) always matches with Guanine(G). The following rules have to be followed :-
1) There should be no sharp turns which implies that the ends of each pair should be separated by atleast 4 intervening bases.
2) The elements in each pair should consist of either {A,U} or {C,G}.
3) No base should appear in more than one pair.
4) There should be no knots which means that if (i,j) and (k,l) are 2 pairs then we cannot have i < j < k < l.
