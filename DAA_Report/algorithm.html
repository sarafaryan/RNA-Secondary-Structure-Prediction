RNA Folding Algorithm

Data Structures -
DP Table - Stores value for every (i,j)
Previous table - Stores the predecessor of the current state.
The RNA Folding Algorithm involves a DP table where every state dp[i][j] is defined as the maximum number of base pairs in a secondary structure for the range bi , bi+1 , ...... , bj.
For every state the optimal value will be the one of the following :-
  1. If j is not a member of a pair, use dp[i][j-1]
  2. If j pairs with some t < j - 4, then we will get 2 independent subproblems dp[i][t-1] and dp[t+1][j-1].
